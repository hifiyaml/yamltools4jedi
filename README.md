# yamltools4jedi
Based on [`hifiyaml`](https://pypi.org/project/hifiyaml/), which is a High Fidelity WYSIWYG YAML parser and emitter than exactly preserve exact formatting, `yamltools4jedi` helps users easily read and edit super YAML files (which may contain tens of thousands of lines) by focusing on small, targeted blocks one at a time—making the process less error-prone.    

Check this [guide](https://github.com/hifiyaml/yamltools4jedi/wiki/yamltools4jedi-guide) for more details

# Installation
```
pip install yamltools4jedi
```
